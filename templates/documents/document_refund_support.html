환불 안내

Preprint는 무인프린트(현장 출력)와 예약 서비스를 함께 운영하고 있습니다.
아래의 환불 조건을 참고해 주세요.

무인프린트 환불 안내

1) 인쇄

환불 대상
  • 결제 후 출력물이 나오지 않은 경우
  • 용지 부족, 종이 걸림으로 인해 미출력된 경우
  • 출력물 색상이 고르지 않게 나온 경우
  • 출력물에 오염 발생(줄, 색상 번짐 등)
  • 종이가 구겨지거나 찝혀서 출력된 경우
  • 이면지 등에 출력된 경우
  • 결제 후 백지가 출력되어 매장에 반납한 경우
환불 불가
  • 컬러 설정을 하지 않아 흑백으로 출력된 경우
  • 인쇄 옵션(용지 크기, 방향 등) 설정 오류로 잘린 출력물 발생
  • PIN 번호 오류로 타인의 출력물이 함께 결제된 경우
  • 복합기 최종 금액 확인 창의 정보를 확인하지 않고 결제한 경우

2) 복사

환불 대상
  • 결제 후 출력물이 나오지 않은 경우
  • 용지 부족, 종이 걸림으로 인해 미출력된 경우
  • 복사물 색상이 고르지 않게 나온 경우
  • 복사물에 오염 발생
  • 종이가 구겨지거나 찝혀서 출력된 경우
  • 이면지 등에 출력된 경우
환불 불가
  • 복사 매수를 잘못 입력한 경우
  • 원본을 잘못 위치시켜 백지가 출력된 경우
  • 원본 방향 설정 오류로 일부 내용이 잘린 경우

3) 스캔

환불 대상
  • 결제 후 스캔 보관함 폴더가 열리지 않는 경우
  • PC 재시작 후에도 스캔 보관함 폴더가 생성되지 않은 경우
환불 불가
  • 원본 방향 오류로 백지가 스캔된 경우
  • 이미지 방향 오류로 일부 내용이 잘린 경우
  • 스캔 보관함에 저장된 파일을 30분 이상 방치해 자동 삭제된 경우
  • 완료된 스캔 파일을 고객이 직접 삭제한 경우

예약 서비스 환불 안내

환불 대상
  • Preprint 서비스 자체 기술적 문제로 인한 출력 오류
  • 네트워크·통신 오류로 전송이 실패한 경우(예: 응답 없음 등)
  • 부분 전송 실패 시, 전송되지 않은 매수에 한해 환불 가능
환불 불가
  • 팝업 공지 확인 후에도 복합기 오류 상태에서 출력을 진행한 경우
  • 원본 위치 오류로 백지가 전송된 경우
  • 원본 스캔 전 결제하여 백지가 전송된 경우
  • 이미지 방향 오류로 일부 내용이 잘린 경우
  • 대량 전송 시 매장 고지(20매씩 분할 발송) 지침을 지키지 않고 20매 초과 발송한 경우

환불 신청 방법

환불 신청은 결제 후 영업일 2일내 전액환불 가능합니다.
환불 신청을 원하시는 경우, 아래 카카오톡 채널을 통해 자료와 함께 문의해 주시기 바랍니다.
쉽고 빠르게 환불 처리 도와드리겠습니다!
